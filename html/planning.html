    PLANNING CHECKLIST

  • - - - INITIATION CHECKLIST - - -

    INPUT:

    COMMON MANUFACTURING EXECUTION SYSTEM (Assigned to you and print Non-Conformance Record as needed)
    PRODUCT DATA MANAGER (Production assist parts and print Manufacturing Change Level Description information)
    VELOCITY (Put Emergency Part Request or Emergent Process Document "in work" and print as needed)
    BLANKET REQUEST (New or "and-on" requirement)

    DATABASE:

    QUICK RESPONSE DATABASE (Create new order and route to Manufacturing Engineer)

    CREATION:

    COMPUTER AIDED PROCESS PLANNING II (Create new plan, update drawings, etc. for "and-on" requirements)
    PROGRAMMING REQUEST (Send email to GRP OAS NC, CC GRP Site Control EO-E (632))
    PHOTO CONTACT MASTER (Order/reserve from Cribmaster)
    MATERIAL REQUEST (Send copy to GRP 632/616 SCMA CC: GRP ResExp; GRP Site Control EO-E (632))
    - - - COMPLETION CHECKLIST - - -

    INPUT:

    COMMON MANUFACTURING EXECUTION SYSTEM (Add routing note and complete/send to next queue)
    - For tag line "SA", put the OID_"NUMBER" in the Alt Acceptance
    PRODUCT DATA MANAGER (Accept part at company for production support and CMES parts with Manufacturing Change Level Description)
    VELOCITY (Enter order number and complete Emergency Part Request)
    BLANKET REQUEST (Include a copy in the package)

    DATABASE:

    PLAN RECORD (Daily input Excel file (1st shift site controller duty))
    QUICK RESPONSE DATABASE (Complete, route, and deliver)
    - Fab blankets to assembly with the Routing Note “blanket shop”
    - Rework blankets and everything else, to staging warehouse

    CREATION:

    COMPUTER AIDED PROCESS PLANNING II (Save complete)
    REQUESTS (Print and include copy in package for Photo Contact Master or Material Order)
    DELIVER (Packet to the checker)
